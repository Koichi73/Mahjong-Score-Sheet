# Mahjong Score Sheet
Mahjong-Score-Sheetは、麻雀の成績を管理するための得点表です。
スプレッドシートを用い、誰でも簡単に使用することができます。

- 1半荘終わりの得点から自動で清算し、
- 当日成績と総合成績に反映する。
- また、総合順位やトップ率なども一目でわかる。
